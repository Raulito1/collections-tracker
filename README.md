# Seso Collection Tracker

A modern financial collections tracking dashboard integrating QuickBooks, Google Sheets, and Notion.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.
